Quest 07

Vectors in Plane

Direction + Magnitude. Master the math behind GPS navigation, game physics, drone flight, and force analysis.

🎭 The Hook

The Drone Navigation Problem

A drone needs to fly from point A to point B while wind pushes it sideways. How does it calculate the correct path?

If the drone's engine pushes it north at 10 m/s but the wind blows east at 3 m/s, the drone doesn't end up going straight north. It drifts. The actual path, speed, and direction are determined by vectors — quantities that carry both magnitude and direction.

Real-World Vectors: Flight navigation uses vector addition to correct for crosswinds. GPS systems use position vectors to pinpoint locations. Game engines use vectors for every moving object on screen.

💡 Discovery Zone

Concepts Unlocked

1 Scalar vs Vector Quantities ▼
Scalar: A quantity with magnitude only (e.g., mass, time, speed, distance, area, volume).
Vector: A quantity with both magnitude and direction (e.g., displacement, velocity, force, acceleration, weight).

Speed is a scalar: "60 km/h". Velocity is a vector: "60 km/h heading north". The direction makes all the difference.

Geometrically, a vector is represented by a directed line segment (an arrow). The length represents magnitude; the arrowhead indicates direction. If a vector goes from point $A$ (tail) to point $B$ (head), we write it as $\overrightarrow{AB}$.

Vectors are denoted by bold letters like $\mathbf{a}$, $\mathbf{b}$, or with arrows: $\vec{a}$, $\vec{b}$. The magnitude is written as $|\vec{a}|$ or $\|\vec{a}\|$.
2 Magnitude (Norm) of a Vector ▼

If $\vec{u} = [x, y]$, the magnitude (length) is:

$$|\vec{u}| = \sqrt{x^2 + y^2}$$

Properties:

  • $|\vec{u}| \geq 0$ for any vector $\vec{u}$
  • $|\vec{u}| = 0$ if and only if $\vec{u} = \vec{0}$
  • $|c\vec{u}| = |c| \cdot |\vec{u}|$ where $c$ is a scalar
  • $|-\vec{u}| = |\vec{u}|$

Example

Find the magnitude of $\vec{u} = [-5, 12]$.

$|\vec{u}| = \sqrt{(-5)^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13$ units

3 Unit Vectors & Normalization ▼

A unit vector has magnitude 1 and points in the direction of a given vector. For any non-zero vector $\vec{u}$:

$$\hat{u} = \frac{\vec{u}}{|\vec{u}|}$$

This process is called normalizing the vector.

If $\vec{u} = [x, y]$, then:

$$\hat{u} = \left[\frac{x}{\sqrt{x^2+y^2}},\; \frac{y}{\sqrt{x^2+y^2}}\right]$$

The standard unit vectors along the $x$-axis and $y$-axis are $\hat{i} = [1, 0]$ and $\hat{j} = [0, 1]$. Any vector in the plane can be written as:

$$\vec{u} = x\hat{i} + y\hat{j}$$

Example

Find the unit vector of $\vec{u} = [-5, 12]$ (magnitude 13 from above).

$\hat{u} = \frac{1}{13}[-5, 12] = \left[-\frac{5}{13},\; \frac{12}{13}\right]$

4 Vector Addition, Subtraction & Scalar Multiplication ▼

If $\vec{u} = [x_1, y_1]$ and $\vec{v} = [x_2, y_2]$:

$$\vec{u} + \vec{v} = [x_1 + x_2,\; y_1 + y_2]$$ $$\vec{u} - \vec{v} = [x_1 - x_2,\; y_1 - y_2]$$

Scalar multiplication: If $\lambda$ is a scalar:

$$\lambda\vec{u} = [\lambda x_1,\; \lambda y_1]$$
  • If $\lambda > 0$: same direction, scaled magnitude
  • If $\lambda < 0$: opposite direction, scaled magnitude
  • If $\lambda = 0$: zero vector $\vec{0}$

Graphical methods:

Head-to-Tail Rule (Triangle Law): Place the tail of $\vec{v}$ at the head of $\vec{u}$. The resultant $\vec{u}+\vec{v}$ goes from the tail of $\vec{u}$ to the head of $\vec{v}$.
Parallelogram Law: Place both vectors at the same point. The diagonal of the parallelogram is the resultant.

Vector addition is commutative: $\vec{u}+\vec{v} = \vec{v}+\vec{u}$

Example

If $\vec{a} = [4, 6]$, $\vec{b} = [-3, 1]$, $\vec{c} = [2, -2]$, find $\vec{a} + 2\vec{b} + \vec{c}$.

$= [4,6] + 2[-3,1] + [2,-2] = [4,6] + [-6,2] + [2,-2] = [0, 6]$

$|\vec{a} + 2\vec{b} + \vec{c}| = \sqrt{0^2 + 6^2} = 6$ units

5 Position Vectors & Rectangular Components ▼
Position Vector: The vector from the origin $O$ to a point $P(x,y)$ is the position vector: $\overrightarrow{OP} = x\hat{i} + y\hat{j}$.

If $\vec{a}$ and $\vec{b}$ are position vectors of points $A$ and $B$:

$$\overrightarrow{AB} = \vec{b} - \vec{a}$$

Rectangular Components: For a vector $\vec{F}$ of magnitude $|F|$ making angle $\theta$ with the $x$-axis:

$$F_x = |F|\cos\theta \quad\text{and}\quad F_y = |F|\sin\theta$$

Conversely: $|F| = \sqrt{F_x^2 + F_y^2}$ and $\theta = \tan^{-1}\!\left(\frac{F_y}{F_x}\right)$

Parallel Vectors: $\vec{u} \| \vec{v}$ if $\vec{u} = \lambda\vec{v}$ for some scalar $\lambda$. In components: $\frac{x_1}{x_2} = \frac{y_1}{y_2}$.
Equal Vectors: $\vec{u} = \vec{v}$ iff $x_1 = x_2$ and $y_1 = y_2$.

Example

A ball is thrown at 25 m/s at 45 degrees. Find horizontal and vertical components.

$V_x = 25\cos 45째 = 25 \times 0.707 = 17.68$ m/s

$V_y = 25\sin 45째 = 25 \times 0.707 = 17.68$ m/s

6 Section Formula (Ratio Formula) & Applications ▼

If point $P$ divides line segment $AB$ in the ratio $m:n$ internally, where $\vec{a}$ and $\vec{b}$ are position vectors of $A$ and $B$:

$$\vec{r} = \frac{m\vec{b} + n\vec{a}}{m + n}$$

Midpoint: When $m = n$, $P$ is the midpoint:

$$\vec{r} = \frac{\vec{a} + \vec{b}}{2}$$

Translation: Moving a point $(x, y)$ by vector $[a, b]$ gives $(x+a,\; y+b)$.

Example

Position vectors of $A$ and $B$ are $2\hat{i} - \hat{j}$ and $3\hat{i} + 2\hat{j}$. Find position vector of $P$ dividing $AB$ in ratio $3:4$.

$\vec{r} = \frac{3(3\hat{i}+2\hat{j}) + 4(2\hat{i}-\hat{j})}{3+4} = \frac{9\hat{i}+6\hat{j}+8\hat{i}-4\hat{j}}{7} = \frac{17\hat{i}+2\hat{j}}{7} = \frac{17}{7}\hat{i}+\frac{2}{7}\hat{j}$

Geometry Applications: Vectors can prove that diagonals of a parallelogram bisect each other, find vertices of quadrilaterals, and verify collinearity of points.
